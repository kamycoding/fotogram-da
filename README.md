# Fotogram

Fotogram is a small photo gallery project built with **HTML**, **CSS** and **JavaScript**.  
The goal of the project is to display a responsive image gallery where users can preview photos as thumbnails and open them in a larger detail view.

## Overview

This project was created as part of the Developer Akademie learning path.  
It focuses on DOM manipulation, dynamic rendering with JavaScript, responsive layout, image handling and basic accessibility.

## Features

- Responsive photo gallery
- Images rendered dynamically with JavaScript
- Thumbnail preview for all photos
- Hover effect on images
- Large image detail view
- Previous and next navigation inside the detail view
- Close detail view by clicking the background
- Close detail view with the `Esc` key
- Keyboard-accessible buttons using the `Tab` key
- Optimized image sizes for better loading performance

## Tech Stack

- HTML5
- CSS3
- JavaScript
